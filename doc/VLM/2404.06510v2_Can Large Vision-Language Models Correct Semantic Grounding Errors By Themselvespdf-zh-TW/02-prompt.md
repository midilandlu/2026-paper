# 翻譯任務提示 (Translation Prompt)

## 任務目標
請將這篇探討「視覺語言模型 (VLMs) 如何透過自我修正來提升語義定位能力」的學術文章，翻譯成流暢的繁體中文 (zh-TW)。

## 翻譯風格與目標受眾
- **目標受眾：** 一般讀者 (general)
- **翻譯風格：** 說故事風格 (storytelling)
請打破原文生硬的學術句型，將內容轉化為流暢、引人入勝的敘事風格。我們要像是在為一位對科技有興趣的朋友解釋一項酷炫的新技術一樣，讓讀者能輕鬆理解為何 VLMs 的自我修正是一項重要的突破。

## 術語表 (Glossary)
- Vision-Language Models (VLMs): 視覺語言模型
- Large Language Models (LLMs): 大型語言模型
- Semantic grounding: 語義定位
- Self-correction: 自我修正
- Oracle feedback: 標準答案回饋
- Zero-shot performance: 零樣本表現
- Fine-tuning: 微調
- Panoptic segmentation: 全景分割
- Chain-of-Thought (CoT): 思維鏈
- Prompt: 提示詞

## 翻譯指引與挑戰
1. **重寫而非直譯 (Rewrite, not translate)：** 使用地道的繁體中文句型，勇敢拆解長句。品質測試：「讀起來像是一開始就用中文寫的文章嗎？」
2. **準確性至上 (Accuracy first)：** 事實、數據、邏輯必須與原文完全一致。
3. **適度解釋 (Proactive interpretation)：** 因為目標受眾是一般讀者，請在第一次出現關鍵概念（如語義定位、標準答案回饋、零樣本表現等）時加上精簡註解，使用 `（**解釋**）` 的格式。例如：語義定位（**將文字描述精準對應到圖像中具體位置的能力**）。但請保持克制，避免過多註解打斷閱讀節奏。
4. **保留格式 (Preserve format)：** 請保留所有的 Markdown 格式，包含標題、粗體、斜體、連結等。
5. **語氣流暢：** 利用生動的措辭與平滑的轉折詞，將枯燥的實驗結果包裝成一段令人著迷的探索旅程。
