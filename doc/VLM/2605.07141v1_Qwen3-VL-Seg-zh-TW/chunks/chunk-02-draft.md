### 4.2 ORS-Bench

#### 4.2.1 Overview
為了全面評估模型在開放世界參照分割（**能根據任意自然語言描述，精準描繪出圖像中對應物體輪廓的能力**）任務上的表現，我們打造了一套高品質的評估套件：ORS-Bench。這套件包含了 ORS-ID-Bench 和 ORS-OOD-Bench 兩個子集。把它們想像成兩場不同難度的考試，前者用來測試模型在分佈內 (ID) 的熟悉場景下表現如何，後者則像是一場突擊測驗，考驗模型在分佈外 (OOD) 的陌生環境中，是否還能舉一反三、展現強大的泛化能力。

#### 4.2.2 ORS-ID-Bench: In-Distribution Benchmark
首先登場的是分佈內基準測試 (ORS-ID-Bench)。它的資料建構方式與我們訓練模型時的流程一模一樣。這裡面包含了 9,055 個高品質樣本，並依據指令類型分為四大類：單一實體類別指令（2,465 個）、多實體類別指令（1,823 個）、片語指令（2,946 個）以及描述性指令（1,821 個）。

這些基於「類別」的指令，是我們從公開的 COCO 與 LVIS 資料集中挑選，再加上我們自建的 SA1B-CoRS 資料集補充而來。至於「片語」指令，則是從 RefCOCO、RefCOCO+ 和 RefCOCOg 這幾個知名資料集的驗證集中精挑細選。而最考驗理解力的「描述性」指令，則源自我們獨家打造的 DeRS 資料集。

為了確保這場考試絕對公平且具備公信力，每一個樣本都經過了嚴格的人工查核。我們不僅要確保像素級遮罩（**在像素層級精確標示出目標物體邊界的遮罩**）標註得完美無瑕，還要保證文字指令與目標物體之間的語義完全吻合。透過涵蓋各式各樣的指令格式與嚴謹驗證的資料，這個基準測試確保了評估過程的真實性與代表性。

#### 4.2.3 ORS-OOD-Bench: Out-of-Distribution Benchmark
為了進一步探索我們模型的極限、應用範圍，以及在那些「不容出錯」的高風險場景中到底可不可靠，我們打造了一個分佈外評估集：ORS-OOD-Bench（請參考圖 7）。

這場考試的題目，在訓練階段幾乎或完全沒有出現過。這能幫助我們評估模型在充滿挑戰的真實世界中，面對難以辨識或分割的物體時，還能不能順利過關。為了準備這些「魔王級」的題目，我們首先從訓練集中抽取樣本，並請多模態大型語言模型 (MLLMs) 幫忙在多個維度上標註屬性，藉此描繪出訓練資料的分佈輪廓。我們特別鎖定六個關鍵維度：目標實體類別、實體佔影像面積比例、描述性指令風格、遮擋程度、影像光照強度，以及風險敏感度。

有了這份分佈分析作為指南，我們接著從一個完全沒有交集的備用資料庫中，挑選出在這六個維度上與訓練資料有著「巨大落差」的樣本。這些樣本包含了訓練時極少見或根本不存在的資料類型（也就是 OOD 實體），而且它們也都經過了嚴格的人工驗證，確保文字指令和遮罩標註精確無誤。

透過這套流程，我們為六種 OOD 類別各精心挑選了 200 個樣本，分別是：類別 OOD、面積 OOD、指令 OOD、遮擋 OOD、光照 OOD 以及風險敏感場景。具體來說：

*   **類別 OOD：** 我們分析了訓練集中的物體類別，並挑選出那些根本沒出現過，或是出現頻率低於 10^-4 的罕見類別來作為考題。
*   **面積 OOD：** 我們專門挑選那些在畫面上極小（面積比例 < 0.002）或極大（面積比例 > 0.7）的目標。
*   **指令 OOD：** 我們調整了 MLLM 生成指令的方式，創造出五種超刁鑽的風格：（1）不直接點名目標的迂迴描述；（2）帶有否定或排除條件的指令；（3）提問式或互動式的查詢；（4）需要複雜空間推理或物體間關係推理的提示；（5）把目標藏在長篇大論背景故事裡的描述。
*   **遮擋 OOD：** 我們優先挑選那些被其他物體嚴重遮擋的目標。
*   **光照 OOD：** 我們選擇了在低光源或夜間環境下拍攝的影像。
*   **風險敏感場景：** 我們納入了自動駕駛和醫療診斷等領域的影像，這些在訓練資料中幾乎是找不到的。

我們相信，這套精心設計的考題，能成為評估參照實體分割任務時，最全面且多樣化的 OOD 基準。它不僅能有效測量最頂尖模型的泛化能力與操作極限，也為未來的視覺定位（**將語言描述精確對應到圖像中具體區域的技術**）研究提供了一個標準化的試驗場，推動技術繼續向前邁進。

### 5 Implementation

#### 5.1 Multi-stage Training
我們採用了兩階段訓練策略，目標是讓模型同時學會強大的多模態理解力，以及處理密集預測任務時所需的細緻感知能力。這種循序漸進的訓練方式，能確保模型的推理邏輯與空間感知能力完美配合、相輔相成。

**階段 1：以分割為核心的適應期 (Segmentation-Centric Adaptation)**
第一階段的首要任務，是善用預訓練視覺-語言骨幹模型中本來就有的視覺定位能力，藉此建立起強大的參照分割本領。既然基礎模型已經能根據自然語言指令找出物體位置，我們就把這種「空間直覺」轉換成像素級的精細預測。

具體來說，我們在一個統一的指令遵循框架下，透過 LoRA 技術來微調語言模型，並同時訓練視覺編碼器和遮罩解碼器。模型會使用固定的輸入模板，在混合了公開參照分割資料集與我們自建的 SA1B-ORS 的資料上進行訓練。在訓練過程中，我們會引導模型產出結構化的結果，裡面包含了邊界框 (bbox) 以及遮罩佔位符 (mask placeholders)。接著，遮罩解碼器會將對應的遮罩標記 (mask tokens) 轉換成最終的二元遮罩 (binary masks)。這個階段成功地在粗略的視覺定位與精細的遮罩預測之間搭起了一座橋樑；同時，視覺編碼器的微調也進一步強化了模型處理遮罩相關視覺特徵的能力。

**階段 2：感知與理解的協同強化 (Synergistic Enhancement of Perception and Understanding)**
有了第一階段打下的基礎，第二階段的目標是恢復並進一步強化模型整體的「多模態理解與推理能力」，同時保持並精進剛剛學會的參照分割技巧。

我們首先將第一階段訓練好的 LoRA 權重合併回大型語言模型 (LLM) 骨幹中。接著，我們保持視覺編碼器凍結不變，對 LLM 骨幹和遮罩解碼器進行全面微調 (full fine-tuning)。這階段的訓練資料是個大雜燴，參照分割、一般多模態理解與多模態推理的資料比例被精心調配為 3:1:2。在推理資料方面，我們請出了原版的 Qwen3-VL-Instruct 模型，透過離線蒸餾 (off-policy distillation) 的方式，生成專注於 STEM 領域的高品質資料。這個階段就像是為模型進行了一場大腦升級，不僅提升了以語言為核心的理解與推理能力，也完好保留了第一階段辛苦學來的視覺特徵。

#### 5.2 Instruction Paradigm and Prompt Design
為了讓模型具備應對開放詞彙 (open-vocabulary) 的能力，我們的指令遵循模式嚴格遵守了 Qwen3-VL 的上下文結構。我們採用了 ChatML 格式，這讓多模態輸入能順暢地融入結構化的對話框架中，就像真人在聊天一樣自然。

**指令模板 (Instruction Templates)**
我們為類別等級與基於描述的分割任務，量身打造了不同的專屬模板。根據指令的精細程度，我們提供給模型的提示 (user prompts) 如下：

*   **基於類別的分割：** "Locate and segment every instance that belongs to the following categories `<category>`, report bbox coordinates and masks in JSON format."
*   **基於描述的分割：** "Locate and segment the object that matches the description `<description>` in the image, report bbox coordinates and masks in JSON format."

**結構化輸出格式 (Structured Output Format)**
為了確保模型給出的答案高度結構化且方便機器讀取，我們嚴格限制模型只能產出特定的 JSON 格式。對於每一個找出的實體，模型必須預測三個關鍵屬性：代表標準化 2D 邊界框座標的 `bbox_2d`、代表類別名稱或識別碼的 `label`，以及代表特定佔位符的 `mask`（例如：`<mask_start><mask_token><mask_end>`）。這個佔位符就像是一把鑰匙，能啟動遮罩解碼器，生成最終的二元遮罩。

#### 5.3 Implementation Details
我們選擇了 Qwen3-VL-4B 作為我們堅實的骨幹模型。我們全新引入的遮罩解碼器非常輕巧，參數只有 17M，大約只佔基礎 MLLM 模型大小的 0.4%。這輕量級的設計，讓模型在進行精確的像素級遮罩預測時，依然能保有原本骨幹模型的高效優勢。

在第一階段，我們使用了 rank 為 32 的 LoRA 技術，進行了 10,000 次的疊代訓練。在這段期間，原版的 LLM 權重是被凍結的，只有 LoRA 適配器、視覺編碼器和遮罩解碼器可以被訓練更新。我們設定初始學習率為 1 x 10^-4，並搭配餘弦退火排程。特別的是，我們給了視覺編碼器一個 0.01 倍的差異化學習率；這能幫助它適應分割任務，同時又不會過度優化，保護了預先訓練好的特徵不受破壞。整個模型採用端對端的最佳化方式，使用混合了文字生成損失與分割損失的目標函數；其中分割損失是逐像素二元交叉熵 (BCE) 損失與 DICE 損失的加權總和。

到了第二階段，我們將 LoRA 權重合併回 LLM 骨幹，接著進行 5,000 次疊代的全面微調。這個階段，LLM 骨幹和遮罩解碼器會一起更新，而視覺編碼器則保持凍結。為了進行這最後的精細打磨，我們採用了 7 x 10^-7 這樣保守的學習率，並搭配餘弦衰減排程。

### 6 Experiments

#### 6.1 Experimental Setup
**資料集 (Datasets)。** 我們的訓練旅程從 RefCOCO 系列資料集（包含 RefCOCO、RefCOCO+ 和 RefCOCOg）起步，先讓模型熟悉參照表達式分割 (RES) 和理解 (REC) 任務。接著，為了挑戰開放世界參照分割，我們使用了一個混合訓練集進行進階訓練，裡面包含了兩個公開資料集 LVIS 和 COCO，以及我們自建的 SA1B-ORS 資料集。在驗證階段，我們使用 RefCOCO 系列的驗證/測試集來評估 RES 和 REC 的表現，並使用 ORS-Bench 來檢驗開放世界參照分割的實力。

**評估指標 (Evaluation Metrics)。** 對於參照分割任務，我們使用了幾個關鍵指標：mIoU（所有樣本交並比的平均值）、cIoU（在整個資料集上計算的整體交並比），以及 precision@t（遮罩 IoU 超過特定閾值 t 的樣本百分比，t 在 {0.5, 0.7, 0.9} 之中）。至於 REC 任務，依照過往研究的慣例，只要預測出的邊界框與真實邊界框的 IoU 超過 0.5 (Prec@0.5)，我們就視為預測正確。

值得注意的是，單一實體和多實體的評估方式有些微不同。在單一實體分割中，我們直接拿預測的遮罩和真實遮罩來計算指標。但遇到多實體的情況時，我們得先計算所有預測遮罩和真實遮罩兩兩之間的 IoU 矩陣，然後使用匈牙利演算法找出最佳的一對一配對。配對成功後，再用和單一實體一樣的指標來進行評估。

#### 6.2 Referring Expression Segmentation
我們首先在標準的 RefCOCO、RefCOCO+ 和 RefCOCOg 基準上，使用 cIoU 指標來檢驗 Qwen3-VL-Seg 的實力。如表 1 所示，我們的模型在 8 個評估子集中拿下了 6 個冠軍，在 RefCOCO 上達到了 82.3/83.7/79.1，在 RefCOCO+ 上達到 76.2/80.2/70.8，在 RefCOCOg 上也有 78.2/78.1 的好成績。

如果和那些基於 SAM 的方法相比，Qwen3-VL-Seg 的表現更是亮眼。它完全不需要依賴外部的分割模型，就能持續產出更高品質的遮罩。舉例來說，在 RefCOCO Val 上，它狂勝 LISA 7.4 分；在 RefCOCO+ TestB 上，更是贏了 12.7 分！即便是面對 SegAgent+SAM，它也分別勝出 3.1 分和 5.4 分。與最新 SAM3 的對決也相當引人注目：Qwen3-VL-Seg 在 RefCOCO+ TestB 上從 63.4 躍升至 70.8，在 RefCOCOg Test 上也從 74.0 提升到 78.1。這強烈暗示了一件事：在處理參照表達式分割時，基於 MLLM 的視覺定位和語言理解能力，遠比純分割基礎模型那種泛泛的概念分割能力來得關鍵。

在那些不使用 SAM 的競爭對手中，Qwen3-VL-Seg 依然是全場焦點，表現優於同樣採用 Qwen 家族骨幹的 Unipixel（在 RefCOCO Val 贏了 1.8 分，RefCOCO+ TestA 贏了 1.3 分）。這證明了明確將預測邊界框作為結構化的先驗知識，比把所有東西混在一起進行純像素預測，能提供更有效的空間約束。更棒的是，Qwen3-VL-Seg 不僅準確率更高，而且與 UFO 和 Text4Seg 等 8B 模型相比參數效率極高。這完美展示了我們提出的邊界框引導遮罩解碼器（**利用邊界框資訊來輔助生成精準像素遮罩的機制**），確實能在小巧的 4B 框架內恢復出高品質的遮罩。

#### 6.3 Referring Expression Comprehension
我們接著來看看表 2 的結果。在所有基於 MLLM 的 RES 方法中，Qwen3-VL-Seg 毫無懸念地拿下了最先進的寶座，甚至在面對那些專精視覺任務的全才模型或大型 MLLM 時，也展現出極強的競爭力。和那些視覺全才相比，我們的模型在所有基準測試上都穩定超車 Grounding DINO，並且與強大的 Florence-2 的差距已經縮小到微乎其微。而在通用型 MLLM 分類裡，Qwen3-VL-Seg 展現了紮實的定位功力，在幾個關鍵指標上超越了 InternVL-3.5，並緊追在目前的王者 Youtu-VL 之後。

最令人驚豔的是，Qwen3-VL-Seg 的表現大幅超越了它的基礎骨幹模型 Qwen3-VL。雖然骨幹模型已經提供了基本的定位能力，但我們的結果顯示，參照分割任務就像是一位嚴厲的教練，透過密集訓練進一步磨礪了模型的空間精準度。具體來說，我們觀察到 Prec@0.5 指標有了穩定的成長，例如在 RefCOCO Val 上從 90.7% 進步到 92.7%，在 RefCOCO+ TestB 上更是來了個 6.6% 的大躍進。這說明了一件事：像素等級的訓練，能促使模型在語義和空間之間建立更緊密的連結，透過更精細的特徵優化，有效昇華了骨幹模型與生俱來的定位天賦。

#### 6.4 Open-world Referring Segmentation
**量化評估 (Quantitative Evaluation)。** 表 3 呈現了在 ORS-ID-Bench 上的成績單。這項測試評估了模型在四種不同指令格式（單一實體類別、多實體類別、片語、描述性指令）下的分佈內開放世界參照分割能力。Qwen3-VL-Seg 在 8 項指標中狂攬 7 項第一，在單一實體類別上達到 96.0/97.4（cIoU/P@0.5），多實體類別 93.2/91.6，片語指令 82.8/94.3，而在最考驗理解力的描述性指令上，也繳出了 91.0/91.3 的漂亮成績。這證明了我們的設計不僅在單純的類別定位上很在行，面對那些需要語言理解與遮罩預測高度配合的靈活指令時，同樣游刃有餘。

在類別導向的測試中，SAM3 是最強勁的對手。它在單一實體指令上拿下了 94.2 cIoU 和 97.3 P@0.5，在多實體指令上也有 94.6 cIoU 和 91.4 P@0.5。然而，Qwen3-VL-Seg 在單一實體的所有指標上都擊敗了 SAM3，並在多實體的 P@0.5 指標上奪魁，僅在多實體的 cIoU 上以 1.4 分之差微幅落後。這場對決告訴我們：當指令與視覺概念高度吻合時，大型分割基礎模型依然非常強大；但我們這套原生於 MLLM 並以邊界框引導的設計，完全不需要外掛其他分割器，就能達到甚至超越它們的精準度。

Qwen3-VL-Seg 真正的殺手鐧，展現在重度依賴語言的測試中。在片語指令上，我們的模型將最強的對手遠遠甩在後頭，cIoU 大勝 19.0 分，P@0.5 也贏了 13.6 分。在描述性指令上，優勢更加明顯，分別領先 15.5 分和 13.1 分。反觀通用型 MLLM 和專精分割的模型，一旦遇到無法只用簡單類別名稱指定的目標，表現就開始大幅下滑。我們認為，這個巨大的優勢歸功於 Qwen3-VL-Seg 能夠直接將 MLLM 原本就具備的定位直覺轉換為密集的遮罩，讓語言語義、邊界框層級的定位，以及像素層級的細化，都能在同一個框架下被完美打磨。

**質性評估 (Qualitative Evaluation)。** 圖 8 和圖 9 分別展示了在單一實體和多實體類別指令下的實際視覺效果對比。我們找來了三種不同架構的高手過招：通用型 MLLM（如 Gemini-2.5-pro、Gemini-3-flash、Seed-2.0 Pro）、通用型感知模型（如 UFO 和 Youtu-VL），以及專職分割的 SAM3。由於 UFO 和 Youtu-VL 不支援多實體分割，所以圖 9 省略了它們。

從畫面上可以發現，通用型 MLLM 雖然能大致抓到目標位置，但畫出來的遮罩常常不夠精細，無法勾勒出複雜的邊界，就像 Gemini 和 Seed 處理大象和毯子時的狀況。另一方面，雖然通用型感知模型和專職分割模型通常能畫出更高品質的遮罩，但在面對開放世界中五花八門的類別指令時，有時會抓不準該覆蓋哪些範圍，或對語義理解有偏差（如 UFO 和 Youtu-VL 處理襯衫，或是 SAM3 處理蓄水池和農夫時的失誤）。Qwen3-VL-Seg 巧妙地將強大視覺-語言骨幹的定位能力，與我們獨創的邊界框引導遮罩解碼器結合。因此，它預測出的遮罩不僅在語義上完美契合指令，還能精準保留物體的幾何形狀和邊界細節。這種結合讓它在穩健性和還原度上，都傲視群雄。

圖 10 接著展示了在應對片語和描述性指令時的精彩表現。視覺化結果證明，Qwen3-VL-Seg 不僅擅長空間推理，就連細微零件的定位也難不倒它。當其他模型還在為相對位置（例如「從下數來第二個瓶子」）或次物體等級的描述（例如消防車上「升起的水砲」）苦惱時，我們的模型總能精準鎖定目標，並給出銳利的像素級遮罩。藉由將強大的語言理解力與邊界框引導遮罩解碼器融合，即使在雜亂無章的開放世界場景中尋找微小目標，Qwen3-VL-Seg 依然能產出既符合語義又在幾何上精確的遮罩。

#### 6.5 OOD Referring Segmentation
為了測試 Qwen3-VL-Seg 在面對極端挑戰時的泛化能力，我們在自建的 ORS-OOD-Bench 上舉辦了一場綜合測驗。如圖 11 所示，Qwen3-VL-Seg 在六大 OOD 場景——類別、面積、指令、光照、遮擋以及風險敏感場景——中，分別拿下了 53.49%、59.30%、86.22%、78.9%、83.45% 和 8.64% 的 cIoU 分數。這些成績顯著超越了其他前沿的 MLLM 和通用型感知模型。這證明了即使遇到從未見過的陌生場景或刁鑽考題，我們的方法依然保持著卓越的泛化能力。

不過，雖然我們在 OOD 戰場上稱霸，但所有參賽模型的表現和在熟悉場景 (ORS-ID-Bench) 的表現相比，都有著明顯下滑。這種衰退在自動駕駛和醫療診斷等風險敏感場景中尤其嚴重，幾乎所有模型都在精準識別和分割上吃了敗仗。這個觀察提醒了我們：未來的參照分割模型必須進一步增強其分佈外泛化能力，才能不斷拓展模型在各種真實應用場景中的適應邊界。

#### 6.6 General Multimodal Evaluation
我們想知道這套多階段訓練策略會不會影響模型原本的一般多模態能力。為此，我們對原版的 Qwen3-VL-4B instruct 模型，以及它在階段 1、階段 2 訓練後的變體，進行了一次全方位評估，涵蓋了視覺問答、多模態推理、密集 OCR 理解以及視覺定位。

表 4 揭露了一個與我們第 5.1 節訓練目標不謀而合的階段性趨勢。在經過第一階段以分割為核心的適應期後，模型在 MMBench-EN 和 RefCOCOval 等偏向感知的任務上有所進步，但在幾個需要推理和密集 OCR 的基準上卻出現了退步，這表明密集的分割監督和視覺編碼器微調讓模型過度傾向空間專業化。

幸好，第二階段的訓練成功把平衡拉了回來。藉由重新引入一般多模態理解和推理資料，同時保持視覺編碼器凍結，最終的 Qwen3-VL-Seg 模型在 MMStar、MMMUval、MathVision、MathVistamini、InfoVQAval、DocVQAval、CharXivRQ、RealWorldQA 以及 RefCOCOval 上的成績，都明顯從階段 1 的低谷中反彈。它在 OCR 和定位相關任務上尤其強大，在 InfoVQAval 拿下 81.4 分，CharXivRQ 45.2 分，RealWorldQA 71.2 分，RefCOCOval 更是高達 92.3 分，優於其他同等 4B 規模的對手。整體而言，這些結果證明我們這套兩階段配方成功找回了模型能力的平衡：第一階段注入分割專屬的空間特化能力，第二階段則漂亮地恢復了細緻感知與一般多模態理解之間的完美平衡。

#### 6.7 Ablation studies
**分割架構拆解分析 (Ablation on segmentation architecture)。** 表 5 呈現了我們的分割架構在 RefCOCO、RefCOCO+ 和 RefCOCOg 驗證集上的拆解研究結果。與 Qwen box+SAM 基線模型相比，不管是完整模型還是被簡化的變體，在所有指標上都有大幅度的成長。這再次確認了我們提出的邊界框引導解碼器，確實能有效將 MLLM 輸出的定位資訊轉換為精確的像素級遮罩。值得一提的是，完整模型在嚴格的 P@0.9 指標上大躍進，在 RefCOCO 上從 39.6 進步到 50.2，在 RefCOCO+ 上從 36.7 提升至 47.6，而在 RefCOCOg 則從 35.9 上升到 47.3。

拆解版本的表現進一步釐清了每個組件的貢獻。如果凍結視覺編碼器 (Ours (freeze vit))，效能會一致性地下降，尤其在 P@0.9 這項高門檻指標上（例如在 RefCOCO 上掉了 6.1 分）。這意味著：想學到超越粗略定位的分割導向特徵，調整視覺骨幹是很重要的。同樣地，若移除多尺度空間特徵注入（**融合不同層次的視覺特徵以增強細節捕捉的技術**）機制 (Ours (w/o multivit))，會導致 cIoU 和 P@0.9 持續掉分，這顯示中介的視覺特徵能提供單靠頂層特徵無法賦予的空間細節。

淺層影像分支也持續發揮作用：若移除它 (Ours (w/o image))，在三個資料集上的表現都會受損，證明了直接的高解析度影像線索有助於還原更精確的物體邊界。有趣的是，完整模型和拆解版本之間的落差，在最對邊界品質敏感的 P@0.9 指標上最為明顯。這暗示著，雖然用較少的組件就能實現粗略定位，但要提升遮罩的精密度和完整性，多尺度特徵注入和高解析度影像分支就顯得格外關鍵。整體結果表明，我們提出的組件是相輔相成的，它們的結合帶來了最穩定而出色的分割表現。

### 7 Conclusion
我們隆重推出 Qwen3-VL-Seg，這是一個專為開放世界參照分割所設計的輕量級框架。它成功地將預訓練的 MLLM 從邊界框層級的定位，拓展到像素層級的遮罩預測。我們的核心理念是：將 MLLM 預測出的邊界框當作遮罩解碼的結構化先驗知識。這樣一來，完全不需依賴外部的分割基礎模型，就能實現精準的分割。

為了讓大規模訓練與評估成為可能，我們建構了 SA1B-ORS 訓練資料集（其中包含互補的類別導向和描述性子集），並引進了 ORS-Bench，用以評估模型在開放世界參照分割中，對於分佈內與分佈外場景的掌握度。在參照表達式分割、視覺定位以及開放世界基準等各項實驗中，Qwen3-VL-Seg 在封閉集與開放世界設定下均表現亮眼，特別是在處理語言密集的指令時有著顯著的優勢，並展現了強大的分佈外泛化能力。額外的一般多模態評估也顯示，模型在經過分割導向的適應訓練後，依然能廣泛保有其原本的一般多模態實力。

這些成果強而有力地證明：將 MLLM 的定位輸出重新當作結構化先驗知識來運用，絕對是邁向統一且高效參照分割技術的一條成功捷徑。
