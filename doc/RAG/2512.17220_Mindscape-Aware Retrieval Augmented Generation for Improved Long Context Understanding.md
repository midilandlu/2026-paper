## **摘要** 

人類在理解冗長且複雜的文本時，仰賴對內容整體的語意表徵。如心理學中人類的_具備心智景觀意識的能力_（Mindscape-Aware Capability，**指人類能夠感知並利用整體知識架構的能力**）所揭示，這種全局觀點有助於組織先驗知識、解釋新資訊，並整合散佈在文件各處的證據。然而，目前的檢索增強生成（RAG）系統通常缺乏明確的全局語意引導，導致在處理長上下文（Long context）任務時，難以檢索與整合分散的證據。在本文中，我們提出了具備心智景觀意識的 RAG（Mindscape-Aware RAG, MiA-RAG），這是首個將具備心智景觀意識的檢索與生成，構建為基於大型語言模型（LLM）之 RAG 的統一條件化範式（Conditioning paradigm）的框架。MiA-RAG 透過階層式摘要（Hierarchical summarization）建立心智景觀（Mindscape），並將檢索與生成皆條件化於此全局語意表徵上。這使得檢索器能夠形成豐富的查詢嵌入（Query embeddings），並讓生成器能在連貫的全局上下文中，對檢索到的證據進行推理。我們在多種長上下文與雙語基準測試中，針對基於證據的理解與全局意義構建（Sense-making）評估了 MiA-RAG。結果顯示它穩定超越了基準模型；進一步的分析也表明，它能將局部細節與連貫的全局表徵對齊，從而實現更接近人類的長上下文檢索與推理。

人類的思考本質上依賴於上下文。對於任何已學習的主題、熟悉的情境或正在進行的專案，人類都會在記憶中維持一個全局語意表徵。當相同的主題再次出現時，此全局記憶會被重新活化，賦予人類**具備心智景觀意識的能力**（Mindscape-Aware capability），使其能感知自身知識的大致範圍，並仰賴

檢索增強生成（RAG）（Zhu et al., 2025; Gao et al., 2023; Zhang et al., 2025a）已成為長上下文理解的關鍵策略，它透過從極長的輸入中檢索有用的上下文分塊，進而克服了 LLM 上下文長度有限的問題（Lewis et al., 2020）。然而，目前的 RAG 系統主要基於局部的、證據層級的訊號進行檢索與生成，缺乏如人類般活化全局語意框架的_具備心智景觀意識的_（Mindscape-aware）能力。賦

- **_豐富的理解_**：由對全局語意的意識所支援，能填補缺失的資訊並解析說明不足的含義。 

- **_選擇性檢索_**：使查詢嵌入（Query embeddings）偏向活化主題的概念框架，過濾掉由其他主題引起的歧義。 

- **_整合性推理_**：在全局上下文中解釋檢索結果，以確保連貫的綜合與理解。 

受上述見解啟發，我們提出了首個為基於 LLM 的 RAG 系統配備具備心智景觀意識能力的方法。具體來說，我們透過以階層式方式生成的摘要來近似長文件的全局印象，將其作為全局記憶的外部表徵，藉此解決長上下文的理解問題。將此形式的全局記憶作為額外輸入，我們訓練模型以適應我們全新的**_具備心智景觀意識的 RAG_**（**MiA-RAG**）框架的兩個核心功能： 

- **_具備心智景觀意識的檢索_**：具備心智景觀意識的能力使查詢能在其全局語意上下文中被理解，產生的查詢表徵不僅錨定於主題範圍（_選擇性檢索_），同時也整合了全局上下文資訊（_豐富的理解_）。我們透過一個特別訓練的嵌入模型來實例化這些能力，該模型將全局記憶與查詢皆作為輸入，並學習將全局資訊整合到查詢嵌入中以提升檢索效能。 

- **_具備心智景觀意識的生成_**：僅透過具備上下文資訊的查詢嵌入來增強 RAG 流程會引入新挑戰：由於生成器無法存取全局上下文，它會變得比檢索器更弱。因此，即使檢索到的內容包含正確的證據，生成器也可能誤判檢索資訊的相關性，或無法有效地利用它。為減輕此不對稱性，我們將生成條件化於相同的全局記憶上。透過將全局記憶納入生成器的輸入中，

模型學會在更廣闊的全局上下文中，解釋檢索到的分塊及其與查詢之間的關係（_整合性推理_），進而產生更忠實的推理與答案。 

我們在一系列長上下文理解任務中評估了 MiA-RAG，涵蓋了中英文、不同的領域與文體（例如政府報告、敘事文本）。評估亦涵蓋各種任務格式，包含自由作答問答、選擇題問答與主張驗證（Claim verification），以及不同的 RAG 配置，如原版 RAG 與 GraphRAG（Edge et al., 2024）。如圖 1 總結所示，MiA 系列模型比基準模型更為有效；特別是 MiA-RAG14B 達成了最佳的平均排名，超越了原版的 72B 系統，凸顯了具備心智景觀意識的檢索與生成的益處。 

除了效能提升之外，我們還透過嵌入空間幾何與一個全新的心智景觀一致性指標，分析了 MiA-RAG 的內部機制。結果顯示它內化了全局語意：心智景觀將查詢表徵重塑為朝向全局語意空間，並作為引導注意力進行整合性推理的支架。 

我們的貢獻總結如下： 

(1) 我們將受心理學與神經科學啟發的具備心智景觀意識的思考問題公式化，並提出了首個為 LLM 配備此能力的計算解決方案。 

(2) 我們在多樣的設定下進行了廣泛的實驗，證明了將具備心智景觀意識的能力整合進基於 LLM 之系統的必要性與有效性。 

(3) 我們的深度分析揭示了心智景觀能將查詢表徵的幾何結構與全局語意空間對齊，並作為語意支架來引導注意力，證實了這是對全局上下文的主動內化，而非表層的模式匹配。

**具備上下文意識的嵌入** 我們的 MiA-Emb 與具備上下文意識的檢索（Anthropic, 2024）之研究主題相關。這一系列工作主要專注於產生富含上下文資訊的嵌入。一種直接的方法是使用支援擴展輸入的 LLM，將每個分塊編碼於長上下文視窗中（Chen et al., 2024; Sturua et al., 2024; Nussbaum et al., 2024; Wang et al., 2024a; Lee et al., 2024; Li et al., 2023; Voyage-AI, 2025）。為整

**長上下文壓縮** 為了捕捉長文件的總體語意，近期的工作已超越標記保留或 KV 快取壓縮（Yang et al., 2024; Li et al., 2024; Xiao et al., 2023），邁向摘要式壓縮與全局記憶（Qian et al., 2025; Behrouz et al., 2024）。MemoRAG（Qian et al., 2025）使用壓縮記憶進行明確的規劃，而 LeanRAG（Zhang et al., 2026）則將知識組織成階層式語意圖以進行結構引導的檢索。相比之下，MiA-RAG 學習將心智景觀內化為檢索與生成共用的條件化訊號，避免了中間的搜尋步驟或外部圖形建構。

### **6.2 MiA 嵌入空間的幾何特性** 

延續第 6.1 節，我們進一步檢驗： 

（_H1_）_MiA-Emb 促進了選擇性檢索（Selective Retrieval）。_ 

也就是說，嵌入模型是否會將查詢表示（query representations）朝向當下活躍的書籍主題偏移，從而使其更準確地位於對應文本塊所支援的子空間內。 

**方法** 我們使用 t-SNE (Maaten and Hinton, 2008) 來視覺化查詢與文本塊的嵌入（chunk embeddings）。為了描繪文件的語義結構，我們首先僅對文本塊嵌入進行 t-SNE 擬合，產生一個能反映該文件語義結構的二維流形（manifold）。接著，我們將查詢表示映射到同一個二維空間中，並觀察各個模型在將查詢定位至相應主題相關文本塊區域的能力。 

**結果** 圖 5 顯示了 MiA-Emb 與原版嵌入模型之間明顯的幾何差異。在不同的書籍中，MiA-Emb 始終產生較小的投影角度，這意味著查詢嵌入更靠近由其對應文件所構成的語義子空間。平均而言，MiA-Emb-8B 的投影角度為 37.1°，而 Qwen-Emb-8B 為 43.5°。這證明了基於心智景觀的條件化範式能更有效地引導查詢前往正確的語義區域，並實現更精確的選擇性檢索。

**方法** 為了瞭解摘要資訊如何豐富查詢表示，我們深入檢視在（_H2.1_）節中找出的那些高度關注摘要的層級。我們的目標是評估嵌入詞元是否將其注意力分配給了與查詢在語義上對齊的摘要詞元。如果這樣的注意力恰好出現在檢索效能提升的層級，這就表示 MiA-Emb 透過有目標地整合全局上下文，增強了對查詢的理解。 

**結果** 圖 7 顯示，在對應於檢索增益的層級中，最終的嵌入詞元將其注意力集中在與答案相關實體、事件或地點相符的摘要分塊上。這表明 MiA-Emb 有選擇性地提取語義對齊的全局線索，並將它們整合到查詢表示中，這印證了逐層分析的結果，並確認了 MiA-Emb 確實透過基於摘要的豐富化來增強查詢理解。

## **7 結論** 

受人類在全局心智景觀中詮釋輸入的認知能力啟發，我們提出了 MiA-RAG，一個為基於大型語言模型（LLM）的 RAG 所設計的具備心智景觀意識的框架。MiA-RAG 使用階層式摘要（Hierarchical summarization）作為持久的全局記憶來近似這種全局印象，並以此作為檢索和生成的條件。MiA-RAG 在基於證據的長上下文（Long context）理解和全局意義構建（Sense-making）任務上都取得了強勁的表現。實證分析表明，心智景觀支援了全局語義的查詢理解、選擇性檢索，以及對分散證據的整合式推理。

## **限制** 

雖然 MiA-RAG 在敘事性長上下文問答和推理方面表現強勁，但它依賴於預先計算的全局摘要作為心智景觀的表示，與標準 RAG 相比，這引入了額外的預處理開銷。儘管這項成本每份文件只會產生一次，並且可以分攤到後續的查詢中，但在實際部署時仍是一個需要考量的實務因素。同時，由下而上的階層式設計支援遞增式更新，只需重新生成受影響的路徑，這在動態或不斷演進的文件環境中部分緩解了這個限制。此外，我們的實驗主要集中在敘事性的長上下文理解上，而其他應用場景仍有待探索。儘管如此，在全局意義構建資料集上的強勁表現提供了初步證據，表明 MiA-RAG 可能可以泛化到純敘事問答環境之外。最後，部分的監督訊號（supervision signal）來自於商業 LLM，這可能會引入潛在的偏差或幻覺內容。然而，實證結果表明，在不完美的監督下，具備心智景觀意識的訓練策略依然保持穩健。

## **A 廣泛研究領域中對具備心智景觀意識能力的支援**

**心理學上的支援** 具備心智景觀意識能力的存在可以追溯到基模 (Schema)（基模 (Schema)）的概念 (Bartlett, 1932)，且與模糊痕跡理論（Fuzzy-Trace Theory, FTT; Reyna and Brainerd 1995）相符。基模 (Schema)為熟悉的主題提供了整合的結構，在資訊處理過程中引導注意力並約束詮釋。FTT 進一步提出，人類記憶在兩個互補的層面上編碼經驗：保留表面細節的**字面痕跡（verbatim traces）**，以及捕捉抽象、基於意義結構的**要旨痕跡（gist traces）**。當新資訊與熟悉的主題相關時，要旨層級的表示會被重新啟動，為詮釋、檢索與推理提供全局的語義鷹架。我們具備心智景觀意識的框架可以被視為這種基於要旨的認知在檢索增強推理系統中的一種運算近似（computational approximation）。
